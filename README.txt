주문출고 PRO PWA v40

Cloudflare Pages 업로드 방법:
1. 이 ZIP 파일을 아이폰 Files 앱에서 압축 해제
2. Cloudflare Pages > 정적 파일 업로드
3. 압축 해제된 폴더 안의 파일 전체(index.html, manifest.json, sw.js, icon 파일)를 업로드
4. 배포 완료 후 pages.dev 주소 접속
5. Safari 공유 버튼 > 홈 화면에 추가

중요:
- 시작 파일은 index.html입니다.
- 기존 데이터는 브라우저 localStorage에 저장됩니다.
- 다른 주소로 배포하면 기존 데이터가 이어지지 않을 수 있으니 백업 기능을 사용하세요.
